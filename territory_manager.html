🗺️ 영토 관리 시스템

GeoJSON/OSM 링크를 입력하면 자동으로 변환하여 MongoDB에 저장합니다

🎯 자동 처리 항목

  • GeoJSON 검증 및 변환
  • Bounding Box (bbox) 자동 계산
  • 시간 범위 (start_year, end_year) 자동 설정
  • MongoDB 인덱스 자동 적용
  • 중복 확인 및 업데이트
  • OpenStreetMap 보기: https://www.openstreetmap.org/#map=5/35.69/116.50

🔐 인증

💡 이미 로그인한 경우 자동으로 감지됩니다. 수동 입력 시 readonly 해제 후 붙여넣기
