# 📜 고려만리지도 사관(史館) 시스템 가이드

> "사관의 붓 끝에서 대륙의 강역이 되살아납니다."

고려만리지도는 『고려사』, 『고려도경』 및 고지도와 천문 기록을 바탕으로 왜곡된 역사를 바로잡는 **사관들의 집단지성 연구소**입니다.

---

## 🏛️ 고려 사관(史館) 통합 18단계 직급표

### 재상급 (정1품~종2품) - 경쟁 임명제

최고의 고증 실력을 갖춘 **단 4인**에게만 허락되는 지고한 직위입니다.

| 품계 | 단계 | 관직명 | 필요점수 | 비고 및 역할 |
|:----:|:----:|--------|:--------:|-------------|
| **정1품** | 1 | 감수국사(監修國史) | 5,000점 + 1위 | 시중(侍中) 겸임. **사관의 최고 수장** |
| **종1품** | 2 | 판사관사(判史館事) | 4,300점 + 2위 | 국가의 대사(大事) 기록 최종 승인 |
| **정2품** | 3 | 수국사(修國史) | 3,700점 + 3위 | 평장사 등 겸임. 실록 편찬 총괄. **승인 권한** |
| **종2품** | 4 | 동수국사(同修國史) | 3,100점 + 4위 | 정당문학, 참지정사 등 겸임 |

> ⚠️ **재상급 임명 조건**: 해당 순위 + 최소 점수를 모두 충족해야 합니다.

## 📊 공적 점수 체계

### 점수 획득 방법

| 활동 | 점수 | 설명 |
|------|:----:|------|
| 📍 사료 제출 | **+3점** | 역사 연구 사료를 제출하면 즉시 획득 |
| ✅ 사료 승인 | **+10점** | 제출한 사료가 최종 승인되면 추가 획득 |
| 👍 추천 받음 | **+1점** | 다른 사관으로부터 추천을 받으면 획득 |
| 🔍 검토 수행 | **+5점** | 다른 사관의 사료를 검토하면 획득 (수찬관 이상) |
| ✅ 승인 수행 | **+5점** | 사료를 최종 승인하면 획득 (수국사 이상) |

### 점수 계산 공식

```
총점 = (사료 제출 수 × 3) + (승인된 사료 수 × 10) + 받은 추천 수 + 검토 점수 + 승인 점수
```

## 🔄 사료 처리 흐름

```
┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│   pending   │ ──▶ │  reviewed   │ ──▶ │  approved   │
│  (검토 대기) │     │ (승인 대기)  │     │  (최종 승인) │
└─────────────┘     └─────────────┘     └─────────────┘
       │                   │
       │                   │
       ▼                   ▼
┌─────────────┐     ┌─────────────┐
│  rejected   │     │  rejected   │
│  (검토 거부) │     │  (승인 거부) │
└─────────────┘     └─────────────┘
```

### 상태별 설명

| 상태 | 아이콘 | 설명 |
|------|:------:|------|
| **pending** | ⏳ | 검토 대기 중. 아직 검토자가 확인하지 않음 |
| **reviewed** | 🔍 | 검토 완료. 최종 승인 대기 중 |
| **approved** | ✅ | 최종 승인됨. 지도에 반영 가능 |
| **rejected** | ❌ | 거부됨. 사유가 코멘트로 남음 |

## 👥 권한 체계

### 검토 권한 (수찬관 이상)

정3품 **수찬관(修撰官)** 이상의 직급을 가진 사관은 다른 사관이 제출한 사료를 검토할 수 있습니다.

**검토 가능 직급:**
- 수찬관, 직수찬관, 사관수찬, 시강학사
- 감수국사, 판사관사, 수국사, 동수국사

### 승인 권한 (수국사 이상)

정2품 **수국사(修國史)** 이상의 직급을 가진 사관만 사료를 최종 승인할 수 있습니다.

**승인 가능 직급:**
- 수국사 (정2품)
- 판사관사 (종1품)
- 감수국사 (정1품)

## 🗑️ 사료 삭제 규칙

### 본인 사료 삭제

| 상태 | 삭제 가능 | 점수 변동 |
|------|:--------:|----------|
| pending (검토 대기) | ✅ 가능 | -3점 (제출 점수 차감) |
| reviewed (승인 대기) | ✅ 가능 | -3점 (제출 점수 차감) |
| approved (최종 승인) | ❌ 불가 | - |
| rejected (거부됨) | ✅ 가능 | -3점 (제출 점수 차감) |

> ⚠️ 승인된 사료는 역사적 기록으로 보존되므로 삭제할 수 없습니다.

### 관리자 삭제

관리자(admin, superuser)는 모든 상태의 사료를 삭제할 수 있습니다.

## 📝 사료 제출 가이드

### 역사 연구 사료 제출 양식

| 필드 | 필수 | 설명 |
|------|:----:|------|
| 연도 | ✅ | 해당 역사적 사건의 연도 (예: 918, 1392) |
| 이름/제목 | ✅ | 지명, 인물, 사건 등의 이름 |
| 출전 | ✅ | 사료의 출처 (예: 고려사 지리지, 삼국사기) |
| 내용 | ✅ | 상세한 연구 내용 및 고증 |

### 좋은 사료의 조건

1. **명확한 출처**: 『고려사』, 『고려도경』 등 정확한 문헌 인용
2. **구체적인 근거**: 천문 기록, 지리 정보 등 객관적 증거
3. **논리적 서술**: 주장과 근거가 명확하게 연결
4. **상호 검증**: 복수의 사료로 교차 검증

## 🎯 성장 여정 가이드

### 입문~중급 구간 (0~1,100점)

- 주로 **사료 제출(3점)**과 학습에 집중
- 기초 고증 능력을 키우는 단계
- 목표: 정5품 기거주까지 진급

### 중급~상급 구간 (1,100~2,600점)

- **검토 점수(5점)** 획득 가능 (시강학사부터 검토 권한)
- 학술적 문지기 역할 수행
- 점수 획득 속도 증가

### 상급~재상급 구간 (2,600~5,000점)

- **승인 점수(5점)** 획득 가능 (수국사부터 승인 권한)
- 프로젝트성 공적 참여 기회
- 재상급 진입을 위한 순위 경쟁

## ⚠️ 주의사항

### 허위 보고 금지

근거 없는 정보를 반복적으로 제출할 경우:
- 사료가 거부(rejected)됨
- 반복 시 경고 및 활동 제한 가능

### 실시간 인사 조정

재상급(1~4위) 관직은 **실시간 점수 경쟁**에 따라 조정됩니다.
- 매일 랭킹 기준으로 자동 갱신
- 순위가 떨어지면 직급도 조정됨

### 불멸의 이름

최고위 사관의 업적은 **'대륙 고려 전도 명예의 전당'**에 영구히 기록됩니다.
