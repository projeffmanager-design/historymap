USER GUIDE

역사지도를
읽는 가장 쉬운 방법

원하는 시점을 고르고, 궁금한 대상을 검색한 다음, 필요한 정보만 지도에 표시해 보세요. 처음 방문한 분도 아래 세 단계로 바로 시작할 수 있습니다.

1

시점 선택

화면 아래 시간 막대에서 연도와 월을 정합니다.

2

대상 찾기

검색창에서 지명·국가·인물·사건을 찾습니다.

3

정보 비교

메뉴에서 필요한 지도 레이어를 켜고 살펴봅니다.

이 페이지의 내용 화면 둘러보기 시간 이동 검색과 상세정보 필터와 레이어 지도 보기 방식 사료 참여 문제 해결
01 · START

먼저 화면을 둘러보세요

상단에는 메뉴와 검색, 지도 보기 버튼이 있고 화면 아래에는 시점을 바꾸는 시간 막대가 있습니다. 지도는 드래그하여 이동하고 확대·축소하여 원하는 지역을 살펴볼 수 있습니다.

상단 왼쪽

햄버거 메뉴, 통합 검색과 좌표 이동 기능을 사용합니다.

상단 오른쪽

고지도 선택, 소식, 랭킹과 로그인 기능을 이용합니다.

지도 영역

국가, 도시, 인물 또는 사건 표시를 선택해 상세정보를 확인합니다.

화면 아래

연도·월을 이동하거나 시간의 흐름을 자동으로 재생합니다.

모바일에서는 일부 메뉴가 아이콘으로 축약됩니다. 왼쪽 위 메뉴 버튼과 검색 버튼을 누르면 같은 기능을 이용할 수 있습니다.
02 · TIME

연도와 월을 이동하는 방법

  1. 화면 아래의 시간 막대를 좌우로 움직여 원하는 시점으로 이동합니다.
  2. −10·−1·+1·+10 버튼으로 연도를 빠르게 조정합니다.
  3. 가운데 연도 입력칸에는 A.D. 391 또는 B.C. 108처럼 직접 입력할 수 있습니다.
  4. 월 표시 양쪽의 화살표로 한 달씩 이동합니다.
  5. 재생 버튼을 누르면 시간이 자동으로 흐르며 지도 정보가 시점에 맞춰 바뀝니다. 멈추려면 일시정지를 누릅니다.
경계와 인물의 활동 기간은 월 단위 자료가 부족한 경우 연도 기준으로 표현될 수 있습니다. 정확한 날짜가 확정되지 않은 자료는 개략적인 시점으로 이해해 주세요.
03 · SEARCH

지명·국가·인물·사건 찾기

상단 검색창에 이름의 일부를 입력하면 관련 결과를 함께 찾습니다. 검색 결과를 선택하면 지도상의 관련 위치로 이동하거나 상세정보를 열 수 있습니다.

  1. PC에서는 상단 왼쪽 검색창을, 모바일에서는 돋보기 버튼을 누릅니다.
  2. 예: “경주”, “고구려”, “광개토대왕”처럼 찾으려는 명칭을 입력합니다.
  3. 결과 유형과 시기를 확인한 뒤 원하는 항목을 선택합니다.
  4. 지도에 나타난 국가·도시·인물·사건을 선택해 설명과 관련 기록을 읽습니다.
현대 지명과 역사 지명이 다를 수 있습니다. 결과가 없으면 한글·한자 표기나 다른 시대의 명칭으로도 검색해 보세요.
04 · LAYERS

필터와 표시 레이어 활용하기

왼쪽 위 메뉴를 열면 지도에 보이는 정보를 목적에 맞게 조절할 수 있습니다. 정보가 너무 많이 겹치면 필요한 항목만 남기는 것이 좋습니다.

민족·국가 필터

특정 집단이나 국가에 관련된 정보만 집중해서 살펴봅니다.

기본 정보

성·도시, 영토, 국가명, 행정구역명과 지명을 켜거나 끕니다.

역사 정보

유적, 이벤트, 연대표, 왕, 역사 기록과 인물 레이어를 선택합니다.

환경·생활 정보

강, 자연, 호구수와 자원 등 역사적 공간을 이해하는 보조 정보를 봅니다.

여러 레이어의 자료 시점과 정밀도는 서로 다를 수 있습니다. 지도상의 겹침만으로 직접적인 역사적 관계가 증명되는 것은 아닙니다.
05 · MAP VIEWS

일반·위성·3D·고지도 비교하기

지도 보기 버튼으로 배경 표현을 바꾸거나 상단의 고지도 메뉴에서 옛 지도를 선택할 수 있습니다.

일반 지도

지명과 경계를 선명하게 비교할 때 적합합니다.

위성 지도

산맥, 강과 해안 등 현재의 자연 지형을 함께 살펴봅니다.

3D 지형

고도와 산지, 이동 경로의 공간적 조건을 입체적으로 이해합니다.

고지도

제작 시기와 표현 방식이 다른 옛 지도를 현재 지도와 비교합니다.

고지도는 제작 당시의 지식, 목적과 표현 관습을 반영합니다. 현재의 정밀한 측량 지도와 동일한 축척·방향으로 해석하면 안 됩니다.
06 · PARTICIPATE

사료와 수정 의견 보내기

지도 메뉴의 사료 제출 기능을 통해 관련 기록을 제안할 수 있습니다. 가능하면 다른 이용자가 내용을 확인할 수 있도록 근거를 함께 남겨주세요.

  1. 수정하거나 보완하려는 국가·인물·사건 또는 위치를 구체적으로 적습니다.
  2. 사료명, 저자, 논문·도서명, 발행 정보 또는 확인 가능한 링크를 첨부합니다.
  3. 자료의 원문과 자신의 해석을 구분하고, 논쟁 중인 견해라면 그 사실을 표시합니다.
  4. 제출 내용은 검토 후 반영되며, 근거가 불충분하거나 중복된 제안은 반영되지 않을 수 있습니다.
07 · HELP

지도가 제대로 보이지 않을 때

  1. 인터넷 연결을 확인하고 메뉴의 새로고침을 실행합니다.
  2. 표시 중인 레이어가 너무 많다면 일부를 끈 뒤 다시 확인합니다.
  3. 3D 지도가 느리면 일반 지도나 위성 지도로 전환합니다.
  4. 문제가 계속되면 사용 기기, 브라우저, 발생 시점과 화면을 koreamanri.com@gmail.com으로 보내주세요.
이제 지도를 직접 살펴보세요.

원하는 연도를 선택하고 익숙한 지명부터 검색하면 쉽게 시작할 수 있습니다.
